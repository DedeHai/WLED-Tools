# FX to GIF recorder

A browser-based tool to record currently running FX and convert it to a GIF.

I made this tool to document effects on the WLED knowledge base (https://kno.wled.ge/) but it can of course be used for other purposes

## How to use

Enter the IP of your WLED controller and hit connect, it will immediately display whatever is running on that controller, just like the "peek" function in the UI.

## Installation

No installation required, just open the html file in your favourite browser.
